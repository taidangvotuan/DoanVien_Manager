# QUẢN LÝ ĐOÀN VIÊN
Phần mềm giúp cho việc quản lý thông tin đoàn viên và thu chi trở nên dễ dàng hơn.
## Cách khởi động
**1. Cài đặt Visual Studio Code**
+ Vào trang web này: __https://code.visualstudio.com/download__ rồi nhấn vào Windows để tải xuống.
+ Sau đó, nhấn vào file __VSCodeUserSetup-x64-1.109.5.exe__ vừa tải.
+ Tiếp tục chọn __I accept the agreement__ rồi nhấn __Next__ liên tục 3 lần.
+ Sau đó tích vào __Create a desktop icon__ (nếu chưa tích) rồi nhấn __Next__ và bấm __Install__.
+ Sau khi tải xuống nhấn __Finish__.

**2. Chạy phần mềm Quản lý đoàn viên**
```
npm install
npm run dev
```
## Các tính năng quản lý thông tin đoàn viên
**1. Nhập đoàn viên mới**
**2. Sửa thông tin đoàn viên**
**3. Xóa đoàn viên**
**4. Tìm kiếm đoàn viên**
**5. Xuất danh sách**

## Các tính năng quản lý thu chi đoàn phí
**1. Nhập thông tin 1 khoản thu/ chi mới (chỉ áp dụng cho mục "Các khoản thuc khác" và "Các khoản chi")**
**2. Sửa thông tin 1 khoản thu/ chi**
**3. Xóa thông tin thu chi (chỉ áp dụng cho mục "Các khoản thuc khác" và "Các khoản chi")**
**4. Danh sách tổng hợp các thu chi của các tháng**
**5. Xuất danh sách thu chi đoàn phí qua file Word**# QuanLyDoanVien
